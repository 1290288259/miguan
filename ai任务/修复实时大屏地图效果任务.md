# 重构大屏实时推送与动态可视化功能任务

## 1. 任务目标
重构前端 Vue 项目中 Dashboard 数据大屏的实时响应相关功能，确保在后端通过 WebSocket 推送最新的攻击数据后，前端能够触发全方位的动态可视化呈现，彻底解决部分动画缺失或无法显示的问题。

## 2. 核心功能需求（重构目标项）
请在重构过程中实现以下四个核心动态效果：
1. **实时地图标注**：只要是识别到是攻击则世界地图上会随着测试请求的发出，在该ip的国家地图地理位置涌现红色的水波纹攻击脉冲点（`effectScatter`）。
2. **报警横幅与弹窗**：识别到带有 SQL 注入载荷等高危攻击时，前台页面右上角必须实时且不断地弹出红色的 [发现高危攻击] 提示横幅或弹窗通知。
3. **数字跳动计数**：Dashboard 总览卡片中的【总攻击次数】等相关统计数值，需要在接收到新攻击时实时递增并伴随跳动动画。
4. **实时告警列表**：在数据大屏的下方或右侧实时攻击列表中，新捕获的 "SQL注入" 等攻击类型记录需要以平滑的动画方式（如滑入动画）不断加载展示。

## 3. 完成标准与测试验收
- 后端持续运行测试脚本 `python test_realtime_report.py` 发送频率极高的模拟恶意 IP 坐标。
- 前端页面能够完美展示上述 4 个核心动态效果，且页面在大流量测试下平滑、不崩溃。
- 测试通过即为完成，过程中不要停止调试，并在验收成功后交付。
- 在测试通过之前，需要不管开发改进，直到测试通过为止。
